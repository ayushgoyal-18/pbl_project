# PUF Research Literature Review

## Overview

This repository contains a curated Excel sheet of 20 research papers related to Physical Unclonable Functions (PUFs). The purpose of this compilation is to support the Project Based Learning (PBL) work titled:

**"Tackling Spurious and Counterfeit Medicines Using PUF-Based Authentication."**

The Excel sheet summarizes key technical and research insights from each paper to enable quick comparison and literature analysis.


## Contents

The Excel file includes the following columns for each paper:

- Author  
- Publish Date  
- Dataset Provided  
- Contribution (Methodology Used)  
- Strength (Issue Resolved)  
- Algorithm Used  
- Evaluation Process  
- Limitations  
- Paper Link  

Each entry has been carefully reviewed and de-duplicated using author name and publication year.

---

## Scope of Papers

The selected papers cover multiple domains of PUF research, including:

- Silicon and hardware PUFs  
- Optical and plasmonic PUFs  
- Chemical and DNA-based PUFs  
- IoT security using PUF  
- Anti-counterfeiting applications in pharmaceuticals  

This progression helps identify the research gap for low-cost, packaging-level authentication of medicines.

---

## Purpose

The main objectives of this compilation are:

- To study the evolution of PUF technologies  
- To compare existing authentication approaches  
- To identify limitations in current anti-counterfeiting solutions  
- To support the proposed PUF-based medicine authentication system  

---

## Usage

Researchers and students can use this Excel sheet to:

- Quickly review important PUF literature  
- Perform comparative analysis  
- Identify research gaps  
- Support academic projects related to hardware security and anti-counterfeiting
